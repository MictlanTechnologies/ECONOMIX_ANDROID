# 💰 ECONOMIX

**ECONOMIX** es una aplicación de **gestión económica personal**, desarrollada principalmente en **Java**, que permite a los usuarios **registrar, analizar y mejorar el manejo de sus ingresos, gastos y ahorros**, combinando herramientas prácticas con contenido educativo.

El proyecto está enfocado principalmente en **estudiantes de nivel medio superior**, pero su diseño permite escalarlo fácilmente a otros públicos.

---

## 📌 Objetivo del proyecto

Los objetivos principales de ECONOMIX es:

- Ayudar al usuario a **visualizar y controlar su economía personal**.
- Promover hábitos financieros saludables mediante **registro, análisis y planeación**.

ECONOMIX no solo registra datos, sino que **enseña a interpretarlos**.

---

## 🧠 ¿Qué problema resuelve?

Muchos estudiantes y jóvenes:
- No saben en qué gastan su dinero.
- No llevan un control de ingresos y gastos.
- No tienen el hábito del ahorro.
- Carecen de herramientas simples para entender su situación financiera.

**ECONOMIX** centraliza esta información y la presenta de forma clara, estructurada y comprensible.

---

## 🏗️ Arquitectura general

El ecosistema ECONOMIX está dividido en varias partes:

### 🔹 Backend
- Servidor desarrollado con **Spring Boot**
- Manejo de datos mediante **Hibernate (JPA)**
- Conexión a base de datos **MySQL**
- Arquitectura REST (API)

### 🔹 Aplicación Android
- Desarrollada en **Android Studio**
- Consumo de API REST
- CRUD completo (crear, leer, actualizar y eliminar datos)
- Interfaz intuitiva y enfocada en el usuario

### 🔹 Base de Datos
- Modelo relacional normalizado
- Tablas para usuarios, ingresos, gastos, ahorros y categorías
- Integridad referencial y consistencia de datos
